# App Downloads Siri Shortcut
A Siri Shortcut to view App Store app download stats and developer country directly from the share sheet, formatted with the myriad numbering system.
## Installation
Download the shortcut via [this iCloud link](https://www.icloud.com/shortcuts/a3fde1047d3a4ddf8fdc15a15be1c97a) or from this repository, and allow access on first run.
## How to Use
1. Open any app page in the **App Store**.
2. Tap the **Share** button.
3. Select **應用程式資訊** from the share sheet.
4. The download statistics and developer country will be displayed in a popup.
